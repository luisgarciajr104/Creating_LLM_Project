# Build a Large Language Model (From Scratch) — Personal Notes

A personal work-through of **[Build a Large Language Model (From Scratch)](https://www.manning.com/books/build-a-large-language-model-from-scratch)** by **Sebastian Raschka** (Manning Publications).

This repository is part of my **portfolio**: it documents my hands-on learning of **PyTorch**, **large language models (LLMs)**, and broader **machine learning / AI** topics as I follow the book.

## What’s here

- Jupyter notebooks and exercises tied to each chapter (e.g. `Chapter2.ipynb`)
- Supporting data and scripts as the project grows

## Tech stack

- **Python**
- **PyTorch** (and related tooling from `requirements.txt`)

## Getting started

1. Clone the repository.
2. Create and activate a virtual environment (recommended):

   ```bash
   python -m venv .venv
   .venv\Scripts\activate   # Windows
   # source .venv/bin/activate  # macOS / Linux
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

> **Note:** Keep your virtual environment **out of Git** (e.g. `.venv/` in `.gitignore`). Recreate the environment on each machine from `requirements.txt`.
